\begin{frame}{Introdução}

\Large{\textbf{Motivação}}
\small{
  \begin{itemize}
         \item Petróleo é uma das principais fontes de energia do mundo;
         \item Exploração e produção se tornou prioridade em muitas economias no mundo;
         \item Exploração tomou conta principalmente dos campos marítimos; %onde se encontra a maior parte das reservas no Brasil
         \item Necessidade de exploração em águas profundas;
         \item Novas tecnologias para fixação e estabilização de plataformas de petróleo no meio do oceano;
         \begin{itemize}
             \item Dutos de captação não possuem resistência suficiente para fixar as plataformas;
             \item Necessidade de um sistema de ancoragem.
         \end{itemize}
         \item 
        
  \end{itemize}
}
\end{frame}

\begin{frame}{Introdução}{Motivação}

Unidades flutuante FPSO:

\begin{itemize}
    \item  Capaz de produzir, armazenar, processar e transferir petróleo de poços localizados a mais de 2000 metros de profundidade;
    \item Pode chegar a uma capacidade de 1,4 milhões de barris de petróleo;
    \item Fixada no fundo do mar por meio de um sistema de amarras.

\begin{frame}{Introdução}


\Large{\textbf{Contextualização do Problema}}\\
\small{


    \begin{itemize}
        \item Plataforma-navio FPSO são capazes de suportar uma vida útil superior a 20 anos;
        \item Relatados casos de rompimento de elos de corrente em menos de 2 anos de operação;
        \begin{itemize}
            \item Falhas nos primeiros elos livres, após a coroa do \textit{fairlead};
            
            \item Atuação de um momento fletor fora dos planos principais dos elos;
            
             \item Projetadas conforme as recomendações da API (\textit{American Petroleum Institute});
            \begin{itemize}
                 \item Normas de dimensionamento dos elos contavam apenas com tensões normais de atuação devido a tração;
            \end{itemize}
        \end{itemize}
    \end{itemize}
}
\end{frame}

\begin{frame}{Introdução}{Contextualização do Problema}

\begin{itemize}
    \item Amarras sofrem com cargas elevadas de serviço, aumentando a força de atrito no contato entre os elos;
    \item Geram um comportamento similar a um engaste;
    \item Atuação de esforços transversais junto do engaste geram os momentos fletores:
    \begin{itemize}
        \item fora do plano de flexão (OPB - \textit{Out of Plane Bending}) %(plano que contém a forma oval)
        \item dentro do plano de flexão (IPB - \textit{In Plane Bending})
    \end{itemize}
    % \item Os momentos fletores maiores estão localizados nos elos próximos a saída do fairlead.
    
\end{itemize}
\end{frame}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{frame}{Introdução}{Contextualização do Problema}

Atuação da carga de prova:
\begin{itemize}
    \item Processo de controle de qualidade de fabricação dos elos;
    \item Carga aplicada de 75\% da MBL (Carga Mínima de Ruptura);
    % \item Eleva a duração da vida de fadiga por tração-tração;
    \item Gera uma deformação plástica que amplia a região de contato dos elos;
    \begin{itemize}
        \item Aumenta a força de atrito e ajuda no comportamento de engaste;
    \end{itemize}
   
\end{itemize}


\end{frame} %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

    \end{itemize}
    
}

\section{Conclusão}
%-------------------------------------------------------
\begin{frame}{Conclusão}


Maior angulação das amarras:
\begin{itemize}
    % \item Plataforma se encontra fora do seu ponto do equilíbrio;
    % \item \textit{Draft} maior.
    \item Maiores as tensões normais no elo;
\end{itemize}    

Definição dos pontos críticos:
\begin{itemize}
    \item Menor resistência à fadiga;
    \item 17° e 30°
    \begin{itemize}
        \item Elo C falharia antes, em comparação ao elo E;
    \end{itemize}
    \item 45° e 60°
    \begin{itemize}
        \item Elo E falharia antes, em comparação ao elo C;
    \end{itemize}
\end{itemize}
A localização dos pontos críticos coincidiu com a localização da simulação.
